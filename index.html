Proyectos destacados

Control escolar familiar: Desarrollador Full Stack

Sistema de asistencia realizado con sistemas de terceros y reconocimiento facial

Objetivo:

Brindar una plataforma para padres de familia que les permita consultar las asistencias de los alumnos en todo momento

Tecnologías clave:

Python Flask, Javascript, Mongodb, HTML y CSS

Objetivos logrados:

  • Manejo de edge cases y errores silenciosos
  • Implementación de base de datos externa y respaldos
  • Optimización en tiempos de proceso de datos
  • Protección a la base de datos contra inserciones y vulnerabilidades
  • Documentación reproducible con Postman

Trash & Track: Desarrollador Backend

Sistema de monitoreo para detección y prevención de riesgos con materiales peligrosos

Objetivo:

Crear una herramienta para mejorar la gestión y manejo de residuos tóxicos

Tecnologías clave:

C++, C#, MongoDB, Mysql, HTML, CSS, TypeScript, React Native

Objetivos logrados:

  • Monitoreo de riesgos en tiempo real
  • Prevención de accidentes en el manejo de materiales peligrosos
  • implementación de nuevas tecnologías en el proceso de recolección

Artemis: Sistema de seguridad

Sistema de seguridad y control de accesos

Objetivo:

Crear una herramienta barata para la protección y monitoreo de negocios

Tecnologías clave:

Mysql, React Native, C++

Objetivos logrados:

  • Monitoreo de rutas de rondines nocturnos
  • Control de acceso por medio de tarjetas RFID asignadas por el administrador
  • Cierre automático de puertas por medio de controles magnéticos
  • envío de reportes automáticos
